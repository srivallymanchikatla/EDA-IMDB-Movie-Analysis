# 🎬 EDA on IMDb Movie Dataset

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on IMDb movie data to uncover insights about ratings, genres, and trends.

## 🛠 Tools Used
- Python
- pandas
- numpy
- matplotlib
- seaborn

## 📊 Key Analysis
- Top 10 highest rated movies
- Most common genres
- Ratings distribution
- Rating vs Revenue relationship
- Movies released per year

## 📈 Key Insights
-Most movies fall between 7.4-8.0 ratings
-Very high-rated movies (>8.5) are rare
-Ratings are concentrated in a narrow range, indicating consistency

## 📁 Files
- eda_imdb.ipynb → Jupyter Notebook
- imdb_movies.csv → Dataset

## 🚀 Conclusion
This project helped in understanding real-world data analysis and visualization techniques.
